9 Box Puzzle Game

Java Swing Sliding Puzzle

A Java Swing-based sliding puzzle with 8 tiles and 1 empty space. Rearrange the tiles in as few moves as possible.

  • Interactive GUI: Enjoy a sleek, responsive interface built with Java Swing.
  • Move Counter: Track every move to solve the puzzle efficiently.
  • Sliding Puzzle Gameplay: Slide adjacent tiles into the empty space to restore order.
  • Solution Checker: Automatically validates your moves upon puzzle completion.
  • Randomized, Solvable Shuffling: Each game starts with a fresh, solvable configuration.
  • Persistent Leaderboard: Records your name and move count in a database.txt file.

Tech Stack: Java, Swing

GitHub: github.com/SamratRay2005/9-box-puzzle
